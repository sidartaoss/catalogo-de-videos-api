# API Catálogo de Vídeos

Propósito:
-
Esta aplicação serve como um backend de gerenciamento para as funcionalidades de catálogo de vídeos.

Tecnologias envolvidas:
-
- JDK 19
- Spring Boot
- Spring Data JPA
- Spring Security
- MySQL
- Swagger
- JUnit
- Mockito
- Docker
- Docker Compose
- Kafka
- Kafka Connect
